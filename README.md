# credit-default-risk

Este proyecto construye y compara modelos de aprendizaje supervisado para predecir el incumplimiento de pago en clientes de tarjetas de crédito. A partir de variables demográficas, historial de atraso y montos facturados, se entrenaron modelos de regresión logística, Random Forest, AdaBoost, XGBoost y redes neuronales, seleccionando como modelo final un XGBoost ponderado que detecta el 65% de los clientes morosos con un AUC-ROC de 0.78 en el conjunto de prueba.
